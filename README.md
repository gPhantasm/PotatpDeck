# PotatpDeck

This mod for **Balatro** adds skins for each suit with art by [Potatp](https://bsky.app/profile/potatpsoup.bsky.social)

## Install

1. Ensure you have [Steamodded](https://github.com/Steamodded/smods) downloaded.
2. Download the repo.
3. Extract to `%AppData%/Balatro/Mods`.
4. Open Balatro.
